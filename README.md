# sub-converter

## Get Started

```sh
git clone https://github.com/uniairport/sub_converter.git
cd sub_converter
pnpm i
pnpm dev
```

## What's this?

In fact, most of the code comes from the [sub-web](https://github.com/CareyWang/sub-web), we have refactored it using Vue 3, Pinia and Vuetify.
